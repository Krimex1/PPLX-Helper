# PPLX Helper

Chrome-расширение для оптимизации работы с Perplexity AI. Добавляет инструменты улучшения промптов и мониторинга токенов.

## Возможности

- Улучшение промптов через OpenRouter API (DeepSeek R1 и др.)
- Встроенный счётчик токенов и ответов за сессию
- Горячая клавиша Alt+Q для быстрого доступа
- Прямая интеграция кнопки улучшения в интерфейс сайта

## Установка

```bash
git clone https://github.com/Krimex1/PPLX-Helper.git
```

### Загрузка в браузер
1. Перейдите в `chrome://extensions/`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку проекта

## Настройка

1. Получите API ключ на [OpenRouter.ai](https://openrouter.ai/)
2. Откройте меню расширения (клик по иконке)
3. Вставьте ключ и выберите модель (например, `deepseek/deepseek-r1:free`)
4. Нажмите **Сохранить**

## Использование

1. Введите текст в поле ввода на [Perplexity.ai](https://www.perplexity.ai/)
2. Нажмите кнопку **Улучшить промпт** в углу поля
3. Дождитесь замены текста на оптимизированный вариант

## Технические данные

**Структура:**
- `manifest.json` — конфигурация (Manifest V3)
- `content.js` — интеграция в интерфейс
- `popup.html/js` — управление настройками
- `background.js` — фоновые задачи

**Разрешения:**
- `storage` — сохранение настроек
- `tabs`, `activeTab` — управление вкладками
- Доступ к `perplexity.ai` и `openrouter.ai`
